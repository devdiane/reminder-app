⏰ Reminder & Notification System

A full-stack automated reminder system that schedules and delivers real-time notifications via Telegram. Built for reliability, background processing, and seamless user automation.

🚀 Live Links
🌐 App: https://reminder-app-mauve.vercel.app/
🤖 Telegram Bot: @reminderOmega_bot
📦 Repo: https://github.com/devdiane/reminder-app

✨ Key Features
🔄 Automated Scheduling – Create reminders that trigger at exact times
📢 Telegram Notifications – Real-time delivery via bot integration
⏰ Background Processing – Reliable execution using server-side jobs / cron
🧠 Time Handling Logic – Smart parsing and scheduling of reminder events
🔁 Reliable Delivery – Retry-safe execution for missed or failed jobs


🛠 Tech Stack
Frontend/Backend: Next.js, Node.js, TypeScript
Database: SQLite / PostgreSQL / Supabase
Automation: Cron jobs, background workers
Integration: Telegram Bot API
Deployment: Vercel
